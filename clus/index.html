MindMeld (https://www.mindmeld.com/) is an open source enterprise conversational platform built by Cisco that enables developers to build deep-domain enterprise conversational assistants (chat or voice).
MindMeld opensource blog post:
https://blogs.cisco.com/developer/mindmeld-ai
MindMeld currently powers Webex Assistant and Cisco.com search. MindMeld is suitable for any enterprise chat and voice applications. Some examples include customer service chatbots in the financial industry, automated appointments scheduler, intelligent employee onboarding assistants and device assistants.
The MindMeld platform is open-source so all data and code belong
to you. Since MindMeld is not a cloud-based platform, it can be integrated with security-conscious customers who have strict data privacy policies. That said, it can be deployed to both cloud and
on-premise infrastructure. MindMeld’s machine learning models have large model capacities and can understand a wide variety of human language while also being easy to configure. MindMeld also
has a native knowledge base built on Elasticsearch for sophisticated information retrieval of business-relevant entities like product identifiers and brand names.
